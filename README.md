# Wedding Portfolio

This is my first client project where I made an effective portfolio for their photography business. It's a beautiful, modern wedding photography portfolio built with React and Vite.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion (animations)
- React Router

## Features

- Modern, responsive design
- Video background
- Image gallery with masonry layout
- Contact form
- Smooth animations
- Mobile-optimized

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/         # Page components
├── assets/        # Images and videos
└── App.jsx        # Main app component
```
